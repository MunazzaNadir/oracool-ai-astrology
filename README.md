# Oracool

Oracool is a personalized AI astrology application built and deployed on Replit.

Users enter their birth date, time, and city to generate a full natal chart. They are then placed into a chat experience with an AI astrologer that can answer questions about relationships, career, personality traits, life direction, and specific planetary placements, with responses tailored to the user’s astrological profile.

## Live Demo

https://cosmic-blueprint-1.replit.app

## Features

- Birth profile input using date, time, and city
- Full natal chart generation
- Interactive AI astrologer chat
- Personalized responses based on astrological profile
- Quick prompt suggestions for easier onboarding
- Premium glassmorphic interface with a purple, pink, and blue gradient design

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Node.js / Express
- Replit deployment

## Project Motivation

Most astrology platforms show charts without making them easy to interpret, while general AI tools are not specialized enough for astrology-based guidance. Oracool bridges that gap by combining astrological profile data with an interactive AI experience.

## Getting Started

Clone the repository:

```bash
git clone https://github.com/MunazzaNadir/oracool-ai-astrology.git
cd oracool-ai-astrology
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```
